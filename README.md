# Customer Feedback Analysis for Cafe Business

## Objective
Clean, analyze, and categorize customer feedback comments to support business insights.

## Project Context
This project is based on a mini assignment simulating customer feedback analysis for a cafe/retail business.

## Tools
- Python
- Jupyter Notebook

## Tasks Performed
- Cleaned raw text data using string methods.
- Standardized customer feedback wording.
- Categorized feedback using conditional logic.
- Summarized results to understand customer sentiment.

## Files
- Project instructions: instructions/mini-project-2-F25.pdf
- Analysis notebook: notebooks/customer_feedback_analysis.ipynb
- Output result: output/customer_feedback_analysis.htm
